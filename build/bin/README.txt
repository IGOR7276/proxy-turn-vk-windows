WDTT v2.0.1 — Windows x64
==========================

1. Распакуйте в любую папку (например C:\wdtt\)
2. Запустите wdtt.exe ОТ ИМЕНИ АДМИНИСТРАТОРА
   (правый клик → "Запуск от имени администратора")

В первый раз:
  • Вкладка "Деплой" → заполните SSH-доступ к VPS → "Развернуть"
  • Вкладка "Туннель" → добавьте сервер через wdtt:// ссылку или вручную
  • Заполните 4 хеша VK → "Подключить"

Файлы:
  • wdtt.exe   — основное приложение (28.5 MB, wintun.dll встроен в exe)
  • README.txt — этот файл

При первом подключении wdtt.exe автоматически извлечёт встроенный
wintun.dll (428 KB) рядом с собой. Если папка с exe недоступна для
записи, dll будет извлечён в %LOCALAPPDATA%\wdtt\wintun.dll.

Изменения в v2.0.1:
  • Дефолт MTU 1280 → 1380 (меньше фрагментации UDP для игр)
  • В Settings добавлены пресеты MTU: 1280 / 1380 / 1420
  • Скрыты мигающие консольные окна при старте туннеля
  • wintun.dll встроен в exe (один файл, без DLL)
  • Профили серверов (сохранение/выбор IP, пароля, хешей)

Подробнее: https://github.com/IGOR7276/proxy-turn-vk-windows
